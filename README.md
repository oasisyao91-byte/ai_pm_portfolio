# AI Product Manager Portfolio - Yao Xiaoyu

This is a single-page portfolio website designed for Yao Xiaoyu, featuring a Tech/AI aesthetic.

## How to View
Simply open the `index.html` file in any modern web browser (Chrome, Edge, Safari, Firefox).
You can double-click the file in your file explorer.

## Features
- **Tech/AI Theme**: Dark mode, neon accents (Cyan/Purple), glassmorphism effects.
- **Responsive**: Adapts to desktop and mobile screens.
- **Interactive**: Scroll reveal animations, hover effects, floating elements.
- **Content**:
  - **Hero**: Introduction and "Open for Opportunities" status.
  - **About/Education**: Education timeline and core competencies.
  - **Experience**: Detailed internship experiences at Tencent, Baidu, iFlytek, etc.
  - **Projects**: Highlights of key AI projects (Agents, Copilots, SFT).
  - **Skills**: Visual representation of technical and product skills.
  - **Awards**: Recent competition achievements.

## Customization
- **Images**: Search for "Place Your Photo Here" in the code or look for the `<img>` tag in the Hero section to replace the placeholder with a real photo.
- **Project Images**: The project cards currently use color gradients and icons. You can replace the `.aspect-w-16` divs with actual `<img>` tags for project screenshots.

## Dependencies
- **Tailwind CSS** (via CDN)
- **Font Awesome** (via CDN)
- **Google Fonts** (via CDN)

No installation required. Internet connection is needed to load the styles and fonts.
