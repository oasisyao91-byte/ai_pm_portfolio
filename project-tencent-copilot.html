腾讯实习 Copilot / LGUI 2025.06 - 2025.09

媒资数据库 Copilot 设计迭代

升级为页面级智能助手 v2.0，引入右侧悬浮气泡、Hover 调用，降低使用门槛，提升工作流融合度。

项目背景

调研发现用户对数据库操作路径高度固化，纯对话式 Agent 使用门槛高；同时 Agent 异步任务耗时较长，需要优化人机交互以保证使用心流，目标提升 Agent 的易用性与工作流融合度。

问题拆解

通过用户调研发现，影响非技术用户使用的核心痛点如下：

  • Prompt 输入理解成本高： 一句话说不清楚需求，AI多轮追问澄清意图。
  • Agent 与页面上下文割裂失焦： “这段对话在聊哪个素材来着？忘了，复制素材 id 回去搜搜。”
  • 长耗时任务管理不标： “素材生产完了吗？手动刷新问一问。”

交互升级

引入右侧悬浮气泡、页面内一键触发、表格 Hover 调用，将数据库管理对话 Copilot 升级为页面级智能助手 v2.0。

项目成果

从纯文本对话到 LGUI Agent

显著降低用户使用门槛与 Prompt 依赖，Agent 深度融入媒资管理流程，成为可持续使用的生产力工具。
