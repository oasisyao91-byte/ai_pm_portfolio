车企销售对话洞察大模型微调 (SFT)

通过微调训练专属小模型，实现车企销售通话录音中 20+ 关键信息的低成本、高并发结构化提取。

SFT Dialogue Insight

项目背景

该车企销售团队通过大量电话触达客户，需从通话录音中结构化提取用户意向、车型偏好、试驾安排等 20+ 关键信息。原有方案依赖人工标注，效率低且成本高。项目目标是通过微调训练专属小模型，实现低成本、高并发的对话数据结构化提取。

实施路径

1. 数据清洗

针对 ASR 语音转文字生成的原始文本，通过正则化与模型介入清洗噪声，修复车企专业术语识别错误。

2. 数据集构建

采用模型蒸馏技术，利用大尺寸模型配合人工标注，构建了约 1000 条高质量样本。

3. 标签平衡

针对 20+ 个关键槽位进行标签分布分析，补充长尾标签数据，避免模型训练过拟合或产生偏见。

4. Prompt 工程

选用 Qwen 系列作为基座模型，设计包含 CoT 思维链与 One-shot 的 Prompt 策略以增强模型推理能力。

5. 模型训练

多轮 SFT 训练，持续调优学习率、Batch Size 等超参；通过 Train Loss 曲线监控训练稳定性，避免过拟合。

6. 效果评估

确立以准确率、F1 召回率为核心的定量指标，结合人工与集外数据测试，评估模型的泛化能力。针对业务特性输出两套模型方案：

  • 方案 A：高准确较低召回
  • 方案 B：高召回可容错

项目成果

F1 Score 提升

基于销售线索不遗漏的业务优先级，业务方最终选择方案 B。
