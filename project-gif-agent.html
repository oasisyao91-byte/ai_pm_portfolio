动图素材生成 Agent

端到端动图生产 Agent，覆盖 60+ 运营用户，整体提效约 90%，生成动图 CTR 高于平均水平。

Project Cover Image

Recommended Size: 1920 x 1080

项目背景与挑战

腾讯视频媒资平台承载全站素材数据，运营在专辑运营中需要大量制作动图封面以提升点击率，但传统流程依赖人工、效率极低。

核心目标： 通过 Agent 改造动图生产流程，实现批量化提效，验证 Agent 在内容生产场景中的业务价值。

业务需求洞察：

  • 数据分析发现动图 CTR 显著高于静态图。
  • 单张制作需 5–10 分钟，具备高 ROI 与人力瓶颈，是理想的 AI 提效场景。

解决方案

1

端到端 Agent 设计

设计全自动化的动图生产 Workflow，串联以下核心节点，大幅减少人工干预：

静态图理解 动图 Prompt 生成 动图生成 回传入库
2

质量优化策略

  • 特征回流机制： 规划基于数据反馈的闭环，提取高转化动图特征（如人物特写、粒子光效）反哺动图提示词生成。
  • 差异化 Prompt 策略： 针对不同素材类型（如动漫、真人综艺）制定定制化 Prompt 模板，确保产出内容的审美与业务场景高度契合。
