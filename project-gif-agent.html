动图素材生成 Agent

从“降本”到“增效”的 AIGC 生产力变革

端到端动图生产 Agent，覆盖 60+ 核心运营。通过“人效压缩”与“质量回流”双引擎，实现生产效率 90 倍提升，点击转化率 (CTR) 超全站平均水平。

Project Cover Image

项目背景：寻找 AI 落地场景

当时我们负责的是腾讯视频的媒体资料数据库管理平台。为响应公司“平台 AI 化改造”的任务，我们需要寻找一个既有业务价值又能落地的场景。经过数据分析，我们发现动图封面的点击率显著高于静图，但生产成本极高，是绝佳的切入点。

Before: 人力瓶颈

运营需手动下载、找工具生成、再上传。一个 S+ 级专辑（近 90 张素材）耗时约 900 分钟（约 1 天）。

After: Agent 自动化

Agent 串联全流程。同等工作量缩短至 10 分钟以内，实现 90 倍人效提升。

解决方案：效率与质量的双重突破

Phase 1: 效率提升（自动化闭环）

我们设计的 Agent 自动串联了整个动图生产流程，运营仅需最后做一次人工确认：

识别静图元素 LLM 生成 Prompt 动图模型生成 自动回传入库

Phase 2: 质量提升（数据驱动）

特征回流机制

Agent 自动分析历史高 CTR 动图（如人物特写、粒子光效），将这些“吸睛特征”结构化回流到 Prompt 生成阶段，让 AI 学会“什么样的动图更吸引人”，实现数据—策略—生成的闭环优化。

分类型生成策略

针对不同素材类型（动漫 vs 真人综艺）调用差异化的 Prompt 模板。动漫强调动作特效，真人强调情绪氛围，确保产出风格统一且贴合内容。
